# Variable Naming Survey — CCSW 325 Course Project

## Instructions for Participants

Thank you for taking part in this study. You will see 25 short Java code snippets. Each snippet is shown twice, side-by-side, with **different variable names** in each version. You do **not** know which version was written by a human and which was generated by an AI; the labels "Version A" and "Version B" are randomly assigned per snippet (the order is shuffled independently for each snippet to avoid bias).

For each snippet, please rate **both versions** on three dimensions, and then indicate **which version you prefer overall**.

### Dimensions

1. **Clarity** — How easy is it to understand what each variable represents from its name alone?
2. **Readability** — How smoothly does the code read overall, given the variable names used?
3. **Meaningfulness** — How well do the names reflect the purpose and intent of the underlying logic?

### Scale

Each dimension is rated on a 5-point Likert scale:

| Score | Meaning |
|-------|---------|
| 1 | Very poor |
| 2 | Poor |
| 3 | Neutral |
| 4 | Good |
| 5 | Excellent |

### Overall Preference

After rating both versions, choose one of the following:

- **Prefer Version A**
- **Prefer Version B**
- **No preference** (the two are roughly equivalent)

## Response Format

For each snippet, fill in the following row in the response sheet:

```
snippet_id, A_clarity, A_readability, A_meaningfulness, B_clarity, B_readability, B_meaningfulness, preference
```

Where `preference` is one of `A`, `B`, or `none`.

## Confidentiality

Your responses are anonymous. Only an anonymized participant ID (P1–P8) will be associated with your answers. The data will be used only for academic purposes within the CCSW 325 course project at the University of Jeddah.
